# Netflix-dashboard using Power BI
## ## Project Overview
This project is an end-to-end Netflix data analysis dashboard built in Power BI. It provides insights into Netflix shows, including genres, ratings, release trends, and show durations.

## Key Features
- Visualize **top genres** and **most popular shows**.
- Analyze **rating distribution** across shows.
- Track **content growth over years** and release trends.
- Interactive **filters for genre, rating, and release year**
