# Netflix-dashboard using Power BI
## Project Overview
The goal of this project was to create an interactive Netflix dashboard using Power BI to analyze and visualize data about Netflix movies and TV shows.

## Insights
1.Dynamic Filters & Slicers
- Interactive filters for Type (Movie/TV Show), Country, Genre, and Release Year.
- Enables users to drill down and analyze Netflix content from different perspectives.

2.KPIs and Summary Cards
- Total Number of Titles, Movies, and TV Shows displayed in visual cards.
- Quick snapshot of overall content volume.

3.Genre & Rating Distribution
- Bar and donut charts showing which genres and ratings dominate Netflix content.
- Helps identify the most popular categories on the platform.

4.Release Year Trend
- Line chart showing number of titles released per year.
- Highlights Netflix’s content growth trend over time.

5.Top Contributing Countries
- Map or column chart visualizing which countries produce the most Netflix titles.
- Useful for geographic insights into Netflix’s content production.

6.Clean & Thematic Dashboard Design
- Designed using Netflix-style color.
- Modern, minimalist layout optimized for readability and user interaction.

## Conclusion
This dashboard demonstrates end-to-end data analysis in Power BI, from data cleaning to creating interactive visuals for actionable insights.
